Types of Linux Shells

Bourne Shell (sh)
Most basic type of shell developed by Stephen Bourne
C Shell (csh)
Syntax is similar to C and it was developed at Berkley. It has features like commands history.
Korn Shell (ks)
This type of shell is like a combination of bourne shell and c shell. It is better than the first two. This was developed at AT&T.
Bourne Again Shell (bash)
This is the most common type of shell which is pre installed in almost every linux distro and many other operating systems.
Z Shell (zsh)
This is an improved version of bash and is the most advanced type of shell out there. It is very popular among developers.
